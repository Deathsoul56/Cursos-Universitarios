## ¿Qué es este proyecto?

Este repositorio contiene apuntes universitarios para carreras de **STEM** (Ciencias, Tecnología, Ingeniería y Matemáticas). Los materiales están diseñados para ser visualizados en **Obsidian**, aprovechando su soporte nativo para Markdown y renderizado de matemáticas via KaTeX/LaTeX.

Los apuntes siguen un formato académico riguroso con:

- Definiciones formales numeradas
- Teoremas con demostraciones completas
- Ejemplos desarrollados paso a paso
- Contexto histórico donde es relevante
- Ejercicios resueltos para práctica

```
Cursos-Universitarios/

│   ├── Clase 1 - Tema X.md    ← Syllabus y temas abordados
│   ├── Clase 2 - Tema Y.md
│   ├── ...
│   ├── Guia 1 - Ejercicios.md
```

## 🛠️ Cómo usar estos apuntes

### En Obsidian (Recomendado)

1. Clona el repositorio en tu carpeta de bóvedas de Obsidian
2. Explora las notas usando los enlaces internos `[[ ]]`

### En otros editores

Las notas son Markdown estándar con sintaxis KaTeX:

- `$expresión$` → matemática inline
- `$$expresión$$` → matemática en display

## 📝 Formato Estándar

Este proyecto sigue una skill de formato académica personalizada definida en `.github/skills/formato/SKILL.md`. Antes de contribuir, lee este archivo para mantener la consistencia.

### Estructura de una sección típica

```markdown
## N. Título de sección

### N.M Nombre del concepto

[Párrafo introductorio]

**Definición N.M (Nombre):**
$$expresión$$

**Teorema N.M:**
$$enunciado$$

**Demostración:**
[desarrollo paso a paso]
$\blacksquare$

**Ejemplo N.M:**
[ejemplo resuelto]

> **Observación:** [nota adicional]
```
